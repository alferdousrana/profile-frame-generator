# 🎯 Profile Frame Generator

A simple and customizable web tool to generate beautiful circular profile frames with text, logo, and user image — perfect for campaigns, events, or personal branding.

🔗 **Live Demo:**
https://alferdousrana.github.io/profile-frame-generator/

---

## ✨ Features

* 📝 Add custom **round text** around the frame
* 🎨 Choose **frame color**
* 🖼 Upload **user image**
* 🏷 Upload **logo** with:

  * Shape (Circle / Square / Rectangle)
  * Adjustable corner radius
  * Optional border (same as frame color)
* 💾 Save configuration (text, color, logo settings)
* 📥 Export frame as:

  * PNG
  * JPG
* 📱 Fully responsive (mobile friendly)

---

## 🚀 How to Use

### 1. Enter Round Text

* Write your desired text in the **Round Text** box
* Maximum **170 characters**
* Character counter is shown (e.g. `45/170`)

---

### 2. Choose Frame Color

* Pick any color using the color picker
* This will apply to:

  * Frame border
  * Logo border (if enabled)

---

### 3. Upload Logo

* Click **Upload Logo**
* Select your image

Then customize:

* **Shape** → Circle / Square / Rectangle
* **Corner Radius** → Adjust with slider
* ✅ **Enable Logo Border** → Adds 2px border

---

### 4. Upload User Image

* Select your profile image
* It will be placed inside the circular frame

⚠️ Note:
User image is **not saved** in local storage

---

### 5. Save Frame Settings

* Click **Save Frame**
* This saves:

  * Text
  * Frame color
  * Logo
  * Logo shape
  * Logo radius
  * Logo border

---

### 6. Export Image

* Click:

  * **Save PNG** → transparent background
  * **Save JPG** → white background

---

## 💡 Notes

* Logo is saved in browser (localStorage)
* User image is temporary (not stored)
* Works fully in browser (no backend required)
